s1-1 · nonlinearity — proof by demonstration

S1-1 · One activation function is the entire difference.

A model with no nonlinearity can only ever carve space with a straight line — it physically cannot separate two concentric rings. Stack one ReLU hidden layer on top of the same inputs, same optimizer, same number of steps, and it wraps the ring cleanly. Nothing else changes.

s1-2 · depth & nonlinearity — proof by demonstration

S1-2 · Depth without nonlinearity is a lie.

Stack five linear layers with no activation between them and the composition is still just one affine map — (W₁W₂W₃W₄W₅)·x + b — so it carries exactly the same expressive power as a single layer. It should fail the ring task identically. Slide a ReLU between each of those same five layers and depth finally does something: now it wraps the ring.
